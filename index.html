Estrutura básica.

(Voltar)

A estrutura básica é formada pelos elementos html, head e por fim body,traduzindo, head significa cabeça e body é corpo, o que faz sentido, já que esse "body" é o corpo do texto que o usuário irá ver.

Tags!

(Voltar)

Existem muitas tags, mas falemos das mais básicas, como por exemplo as tags, "i" ou "strong", eu acabei de usar a tag strong, para deixar o texto mais forte onde eu quero destacar, como por exemplo ao citar as tags "i" e "strong", estranho e divertido não?

Basicamente as tags servem para mostrar ao navegador a estrutura do site! Elas também sempre terminam com uma barrinha, o "/" sinalizando ofim da tag, tudo que abre, fecha, é claro que algumas tags não seguem a regra como no caso da tag "input", essa tag cria um campo para digitar, igual ao que eu fiz no começo para que você pudesse se apresentar!

Atributos das Tags.

(Voltar)

Atributos são nada mais nada menos do que propriedades que uma tag pode ter, existem atributos gerais que todas as tags podem ter e atributos especificos de uma tag.

Como exemplo de atributos temos o "style que muda a cor de um texto, ou o "class" em que você pode por um título padronizado.

Textos

(Voltar)

Quando se escreve em html se faz uso de algumas tags como "h1" "h2" até "h6" que são os cabeçalhos, os títulos, sendo o "h1' o maior e o "h6 o menor".

Então entendendo que os "hs" são usados para títulos usamos o "p" para paragráfos e o "blockquote" para citações, por exemplo:

"O começo de todas as ciências é o espanto de as coisas serem o que são - Aristóteles"

É claro que para deixar a formatação mais bonita eu usei a tag "i" de itálico, mas a citação fica com a formatação própria, apenas usando o "blockquote".

Listas Ordenadas e Não Ordenadas

(Voltar)

O "ol" por exemplo é a lista ordenada, mas ela não funciona sozinha, precisando da tag "filha" o "li", a lista ordenada, a partir dai se coloca os itens, um embaixo do outro.

Agora o "ul" é uma lista não ordenada, é uma lista que leva marcadores na frente, as "bolinhas", ela não tem uma ordem, diferente do "ol" que enumera a lista.

Basicamente nos navegamos na internet o tempo todo e os links nos ajudam com isso, nos levando de um lugar para outro

Usamos o "a" para "acorar" quando estamos indo para algum lugar.

Tags Form

(Voltar)

Usamos essa tag a "form" para criar um formulário, assim você consegue preencher os campos, é muito utilizado em pesquisas e formulários diversos, vou mostrar um exemplo.

Vou fazer um pequeno formulário para que você responda, veja:

Nome:
Idade:
Senha:

Primeiro dentro do body a gente começa com a propria tag "form", você a nomeia, eu nomeei a minha como contact, é importante saber que na programação usamos muito o inglês.

Então eu faço uma linha para cada informação, como nome e idade, usamos o "input" para definir se esse é um campo de texto, número e até senha, e para tornar de fato um formulário usamos a tag "button" e definimos que ali é onde você clicará para enviar.

Mas e agora? Se você clicar em enviar, para onde essa informação irá? É bem simples, na linha da tag form usamos uma tag chamada "action" a partir dai direcionamos essas informações para um site, eu coloquei por exemplo para que vá para meu site pessoal. Eu também posso usar a tag "method "="post"" assim quando você enviar suas informações ela não apareça na barra de url, é uma forma muito mais segura de manter tanto seus dados quanto senha ocultos.

Alias esse formulário é diferente do começo da página, eu usei a tag "input" mas sem o form, como uma brincadeira, ou seja suas informações não foram salvas, essa tag "input" é como se fosse uma tag filha da tag "form" por isso ela abre um campo mas não funciona de fato como um formulário sozinha.

Podemos usar também a tag "target" ela decidirá se a página será aberta em outra aba ou na própria página, "blank" abre em outra página, "self" abre na mesma página. Podemos usar também a tag "autocomplete", basicamente ele completa para você o campo, como quando o navegador pergunta se você tem interesse em manter sua senha salva, no entanto se você decidir que não, usa-se "autocomplete="off"", muito útil não? Muitas dessas tags usamos o dia a dia e nem percebemos.

Por fim tem uma tag muito interessante, a tag "onsubmit", precisamente tudo que começa com "on" é um evento, e ao usar essa tag eu posso enviar um alerta para que quando você terminar o formúlario você saiba que foi definitivamente enviado.

Tag Input e Tipos

(Voltar)

A tag "input" tem muitos tipos, eu usei varias delas para estruturar o site, vamos estudar um pouco mais sobre elas sim?

Toda tag "input" é seguida do "type=""" a partir dai temos varias outras tags, como por exemplo: "text" para texto, ou number para número, vale lembrar que essa ordem de "sub" tags estão em ordem alfabetica.

Também podemos fazer uso da tag label usamos ela dentro de uma tag "form", ela fica parecida com um formulário comum mas a estrutura dentro do body é diferente, basicamente, fica mais ou menos assim:









Usando a tag "label" você digita o que deseja, seguido da tag input e "text" para texto, vale lembrar que se pode usar o campo texto para escrever letras e numeros, diferente do campo number que só pode ser usado para escrever números.

Você pode também, no caso, usar o "min" que significa minimo, para definir o minimo que o tipo numero pode ir, por exemplo excluir a possibilidade de ir para um número menor do que zero. Há também o contrário, max para definir o número máximo que pode ser atingido. Outro atributo bem legal é o "step" com ele você pode pular de uma quantidade determinada, de 2 em 2, de 5 em 5, e por ai vai.

Outra tag que podemos usar é o "button" ou "botão", dentro dele você pode utilizar a tag "value" que gerara um botão na estrutura do site.

Uma tag super interessante é o "range", ele cria uma barra de rolagem, ele tem a mesma estrutura dentro do body, mas a formatação dele é diferente, ele também funciona com as tag "mim" e "max", ou seja ele pode ter um limite.

Outra tag que vale a pena mencionar é a "color" que cria um campo de cor no qual você mesmo pode escolher a cor.

Mas não acaba por ai, temos as tags, "email" para uma caixa de e-mail, temos, "url" para uma caixa de url, há "date" para data.

Select e Seus Tipos

(Voltar)

Usamos o select para selecionar as informações que vão em um campo, por exemplo, quando você está se matriculando em uma faculdade ou buscando emprego, tem um campo perguntando qual área de interesse, mas ao invez de digitar, o botão select já tem criado semi-categorias/opções para que você possa escolher clicando e não escrevendo.

Com a tag "option" você tera um valor vazio com uma frase como: "selecione o cargo" ou "selecione o curso desejado" e então terá as opções, como história, letras, matemática etc.

Textarea

(Voltar)

Em curtas palavras, "textarea" é uma área de texto, essa tag cria um campo de texto geralmente no final de um formúlario, como uma mensagem para o dono do site, ou algo que você quer comunicar, ele é maior que os outros campos de texto e possuiu duas barrinhas no final no canto direito para aumentar a área em que você quer escrever, isso se deve já que essa área de texto é própria para uma consideravel discussão ou comunicado.

Você também pode definir a quantidade de linhas e colunas, seja para um texto mais longo ou com mais paragráfos.

(Voltar)

"O elemento de divisão "div" é um container genérico para conteúdo de fluxo, que de certa forma não representa nada" - https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/div

Tag Img

(Voltar)

A tag "img" é uma tag que permite inserir imagens e gifs no corpo do texto, mas ela não funciona sozinha, sempre que se usa ela, é necessário fazer uso do "src o source ou a fonte da imagem, podendo ser um arquivo da maquina ou da internet.

Outra coisa importante é o "width, diferente do src que vem automaticamente ao digitar a tag img, o widht é opcional, ele serve para alterar o tamanho da imagem, caso você não faça uso dele, a imagem ficará do tamanho original.

É útil falar sobre a diferença entre png e jpg, por exemplo, pgn por exemplo é um formato mais pesado do que jpg, embora o png seja mais adpatavel a diferentes cores, ficando mais fluxo e bonito esteticamente.

Outros atributos interessantes são o "tittle", título e o "alt" ambos podendo ser usados na tag img.

O tittle, adiciona um pequeno balão de informação, enquanto o alt é uma breve descrição, ele é uma ferramenta auxiliadora para uma pessoa que não pode enxergar, servindo como uma descrição que pode ser ouvida:
