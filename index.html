Estrutura básica.

(Voltar)

A estrutura básica é formada pelos elementos html, head e por fim body,traduzindo, head significa cabeça e body é corpo, o que faz sentido, já que esse "body" é o corpo do texto que o usuário irá ver.

Tags!

(Voltar)

Existem muitas tags, mas falemos das mais básicas, como por exemplo as tags, "i" ou "strong", eu acabei de usar a tag strong, para deixar o texto mais forte onde eu quero destacar, como por exemplo ao citar as tags "i" e "strong", estranho e divertido não?

Basicamente as tags servem para mostrar ao navegador a estrutura do site! Elas também sempre terminam com uma barrinha, o "/" sinalizando ofim da tag, tudo que abre, fecha, é claro que algumas tags não seguem a regra como no caso da tag "input", essa tag cria um campo para digitar, igual ao que eu fiz no começo para que você pudesse se apresentar!

Atributos das Tags.

(Voltar)

Atributos são nada mais nada menos do que propriedades que uma tag pode ter, existem atributos gerais que todas as tags podem ter e atributos especificos de uma tag.

Como exemplo de atributos temos o "style que muda a cor de um texto, ou o "class" em que você pode por um título padronizado.

Textos

(Voltar)

Quando se escreve em html se faz uso de algumas tags como "h1" "h2" até "h6" que são os cabeçalhos, os títulos, sendo o "h1' o maior e o "h6 o menor".

Então entendendo que os "hs" são usados para títulos usamos o "p" para paragráfos e o "blockquote" para citações, por exemplo:

"O começo de todas as ciências é o espanto de as coisas serem o que são - Aristóteles"

É claro que para deixar a formatação mais bonita eu usei a tag "i" de itálico, mas a citação fica com a formatação própria, apenas usando o "blockquote".

Listas Ordenadas e Não Ordenadas

(Voltar)

O "ol" por exemplo é a lista ordenada, mas ela não funciona sozinha, precisando da tag "filha" o "li", a lista ordenada, a partir dai se coloca os itens, um embaixo do outro.

Agora o "ul" é uma lista não ordenada, é uma lista que leva marcadores na frente, as "bolinhas", ela não tem uma ordem, diferente do "ol" que enumera a lista.

Links e Navegação

(Voltar)

Basicamente nos navegamos na internet o tempo todo e os links nos ajudam com isso, nos levando de um lugar para outro

Usamos o "a" para "acorar" quando estamos indo para algum lugar.
